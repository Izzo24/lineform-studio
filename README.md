# 線構 LINEFORM STUDIO

沉浸式線條設計工作室形象官網（預覽）。單頁式 WebGL 沉浸體驗。

- **技術棧**：原生 HTML / CSS / JavaScript + Three.js (r128, WebGL)
- **風格**：手繪線條 / 紙感 ink-on-paper / 一筆畫美學
- **互動**：滑鼠視差、捲動驅動的 3D 線框造型變換、自訂墨點游標、生成粒子場
- **內容**：理念、能力、精選作品、合作流程、聯絡
- 所有插畫由 AI 生成（線稿風格），Logo 為自訂 SVG
- 響應式（RWD），繁體中文
